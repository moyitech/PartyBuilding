# PartyBuilding
智慧思政智能体平台
太原理工大学计算机科学与技术学院所开发的智慧思政智能体平台

## 技术架构
前端：微信端Web网页，用于用户在线使用智能体，自适应 PC Web 和手机端 Web。无登录功能。
后端：使用 fastapi
前后端分离，前端通过 static 挂载的方式和后端一起部署

## 主要内容
智能体1:政策智能问答智能体。以聊天的形式和智能体进行对话。

智能体2:常规组织生活设计智能体。部委员输入主题关键词（如“科技创新”、“纪律教育”）、活动时长、参与对象等参数，系统可自动生成一份完整红色代码资源库的、符合“三会一课”规范的组织生活方案草案，包括学习资料、讨论议题、活动流程建议等，极大地减轻了基层党务工作者的负担。在前端输入关键词，然后一键生成，采用端到端、非对话形式进行。

智能体2:沉浸式组织生活设计智能体。“代码长征：重温红色电信史”的沉浸式主题党日。参与者需要通过编写简单的代码来破解“密电码”、完成“信息传递”任务，将计算机专业知识与党史学习深度融合。在前端输入关键词，然后一键生成，采用端到端、非对话形式进行。

智能体3:音乐智能体。输入关键词（如“奋斗”、“青春”），AI可生成一段符合意境的红色旋律或推荐经典红歌并解析其创作背景。在前端输入prompt，然后一键生成，采用端到端、非对话形式进行。其prompt也可点击自动生成。

智能体4:党史情景生成智能体。输入历史事件（如“遵义会议”），AI可生成一段生动的场景描述，并可进一步转化为图文或视频脚本。在前端输入关键词，然后一键生成，采用端到端、非对话形式进行。

智能体5:绘声绘色政策解读智能体。将党的二十大报告中的某一段落输入，AI可自动将其转化为一份H5图文海报、一个1分钟短视频脚本或一套互动测试题。在前端输入关键词，然后一键生成，采用端到端、非对话形式进行。

## 前端风格

整体前端风格以“庄重、简洁、智能化”为核心设计理念，兼顾高校党建场景的严肃性与智能体产品的现代交互体验。

### 设计理念
- **红色主色调 + 现代扁平化设计**：整体色彩以红色系、白色与深色中性色为主，体现党建主题的庄重性，同时避免过度装饰，保持界面简洁克制。
- **内容驱动优先**：界面设计服务于“智能生成内容”这一核心目标，弱化复杂导航与冗余元素，突出输入区、生成按钮与结果展示区域。
- **低学习成本**：面向基层党务工作者与学生用户，界面操作直观清晰，无需使用说明即可完成关键词输入与一键生成。

### 交互风格
- **一键生成，结果导向**：除政策智能问答外，其余智能体均采用端到端、非对话式交互，用户只需输入关键词或参数即可获得完整生成结果。
- **轻交互、强反馈**：通过加载动画、生成进度提示等方式增强“AI正在思考”的感知，提升用户体验与系统可信度。
- **模块化智能体入口**：不同智能体以卡片或模块形式呈现，功能边界清晰，便于用户快速选择所需能力。

### 终端适配
- **微信端 Web 优先设计**：界面尺寸、按钮布局与交互方式针对手机端优化，适合在微信环境中直接使用。
- **PC 与移动端自适应**：采用响应式布局，同一套前端页面可在 PC Web 与手机 Web 上良好展示，无需额外适配。
- **免登录设计**：不引入账号体系，用户可即开即用，降低使用门槛，适合组织生活现场或教学场景快速使用。

### 视觉呈现
- **突出生成内容展示**：生成结果以卡片、分段文本或结构化模块方式呈现，便于直接阅读、复制或后续整理。
- **党建元素适度点缀**：在不干扰使用的前提下，通过图标、标题样式等方式强化党建与思政属性。
- **统一风格规范**：各智能体在视觉风格与交互逻辑上保持一致，形成统一的“智慧思政智能体平台”产品形象。
